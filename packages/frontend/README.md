# vue-caculator

## Getting started
- Clone your this repo
- Install node dependencies
```Bash
cd path/to/your/project/moorepo_caculator/packages/frontend/
yarn
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
